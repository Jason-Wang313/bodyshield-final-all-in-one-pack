# GitHub Repository Note

This public repository tracks the BodyShield non-hardware source, paper draft,
reports, compact result artifacts, and the compressed release bundle.

The generated `results/trials.csv` file is intentionally not tracked directly
because it is about 344 MB, above GitHub's ordinary file-size limit. The full
raw trials table is preserved inside:

`release/bodyshield_non_hardware_release.zip`

For the final verification state, see:

- `reports/NON_HARDWARE_COMPLETE.md`
- `reports/PACK_VERIFICATION.md`
- `reports/RELEASE_BUNDLE.md`
